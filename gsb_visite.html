Visite

Présentation:

Le projet GSB visite consistait en créer une application Android pour les visiteurs médicaux leurs permettant de gérer leurs différentes visites. Nous trouvions sur cette application un système permettant de créer des visites, faire des rapports sur ceux-ci, retrouver ses anciennes visites, voir des bilans de celle-ci avec le professionnel de santé visité, le but de cette visite, si un échantillon a été laissé...

Création:

Ce projet a été créé en différentes parties. Tout d'abord, nous avons dû créer la base de données correspondante au diagramme de classe que nous avions dressé au part avant. La base de données a été créer sur MongoDB Atlas ce qui en fais une base de données NoSQL, ce qui fait que les tables ne sont pas en relation. Nous avons ensuite dû créer une API REST en JavaScript qui sera accessible par notre application Android. Cette application a été dévelppée en Java avec AndroidStudio pour la rendre utilsable par les visiteurs.

Choses Appris
Apprentissage de nouvelles notions Avant ce projet, je n'avais jamais créer ni utilisé d'API REST. J'ai donc dû apprendre à créer une API REST en JavaScript. J'ai aussi dû apprendre à utiliser une base de données NoSQL avec MongoDB Atlas. J'ai donc dû apprendre à utiliser cette base de données, à la créer et à l'utiliser dans un projet.
Comprendre de nouveaux langages Ce projet à nécéssité l'utilisation de nouveaux langages. J'ai donc dû apprendre à utiliser le langage JavaScript pour la création de l'API REST. J'ai aussi dû apprendre à utiliser le langage Java pour la création de l'application Android.
